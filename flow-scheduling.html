Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

Project : Lightning Flows Scheduling

CASE :It is required that flows should be Scheduled to run for a batch of records at night daily to update new leads without email and set their status to Invalid.
Create a web to lead form with the following fields

  • First Name
  • LastName
  • Email
  • Phone
  • Company
  • City
  • Province/State
and for every incoming lead ,schedule a flow to check if email is blank and set the status to invalid.

  1. Got to setup, web-to-lead ,click on create Web to lead form. Include all the above fields and set a return url. Then click on generate.


    • Create the Schedule-triggered Flow and set the time to run as required


    • Choose starting object ,loop over the records, assign these records to a single variable and reassign them again




    • then update the list of lead records


    • To see the scheduled flow, go to setup,scheduled job
