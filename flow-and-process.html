Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

Updating Child record field from parent

CASE : Update Opportunity primary contact field anytime account primary contact is updated

Result :



  1. Go to setup,flow,Autolaunch flow.

  2. Create a new resource variable with datatype text to store account record id which will be passed through process builder.


  3. Get opportunity with account id equal to the above id .You can choose fields but in this case i will go with storing all fields

  4. Loop over this opportunity records and assign them to a new resource variable text.This will be used to store the primary contact of opportunity


  5. Add an update element to update the primary contact on all looped opportunities.


  6. Create a process to pass the account record id and primary contact .


  7. Activate and Test
