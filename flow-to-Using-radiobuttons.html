Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

This screen flow provide a quick way to create records and relate to Parent records without neccessarily creating from record page.


Result :



  1. Go to setup,flow,Autolaunch flow.
  2. Create a record choiceset variable


  3. Add a screen component.Within the screen component add a radio button and let choice be the variable created before. Add different input fields by data types you wish to collect when creating the opportunity record.

  4. Add a new element to create the opportunity with information from the screen component. AccountId will be passed from the choiceset variable while other fields such as stage,opportunity name and date from screen inout component.


  5. Debug your component and later activate. Go to app builder to either add to page or use as quick action depending on your use case.
  6. Activate and Test
