Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

Project : Lightning Flows and LWC

CASE : As a fast paced sales company where every minute counts,my team is stressed that they forget fields they need to fill out when booking an opportunity to closed/Won and they find valdation rules too complicated. So our manager requires both a declarative and programmatic solution to choose from.

  1. Build out a screen flow that captures the data required when an opportunity is moved to closed won.
  2. Once flow is complete, an automated email should be sent to the opportunity owner's manager letting them know that the opportunity was booked and Display the following data as required for moving to closed won:
    • Monthly shipment
    • primary contact
    • expected award date
    • Short description
    • Expiration date
  3. Automatically launcha LWC and request the field above (2) and make sure all are required. Basically same criteria as above

Result : Click Image To View



  1. Create a Custom object named Account Address with required fields

    • Go to Setup,Object Manager and create custom object with required fields and data types or use the schema builder
    • Create a master detail relationship between this object(account Adddress object) and Account.Then proceed to create all require fields
    • Main flag field should be a formula field . It will be used to display our flag icon in the datatable

  2. Write Apex Classes for Account and Account Address objects to meet our requirements


    • We need an apex class accountHandler for account


    • We need another apex class myAccountAddressHandler for Account Address cusom object

  3. Write Apex trigger for various trigger events

    • Apex trigger for myAccountHandler Class

    • Apex trigger for myAccountAddressHandler Class

  4. Create Ligthning Web Component to Display Account Address records related to each account with main flag

    • First create an apex class . This class contains 2 methods to get all fields for our lwc datatable and modal pop-up and a second method to delete list of records

    • Create LWC HTML

      this will contain our datatable and modal for our action buttons


    • Create LWC js

      Main flag is a formula field which is used to flag the address type if it is main or not
