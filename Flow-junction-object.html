Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

Flow and process to send Opportunity Contact roles Emails on Opportunity stage change

CASE :As opportunity stage changes , its required for all the opportuning contact roles to receive an email notifying each of them.

Result



  1. First we need to create an email template for Opportunity contact roles and add to email alert. This will be used later in the flow action.Note certain fields like the contact first name can't be referenced on the opportunity contact role object so use formula fields .
  2. Go to setup an search flow.choose autolaunch.
  3. Create a record variable to store the list of Opportunity Contact roles . This list of contact roles will be passed in our process builder.Choose available for input(this make it available to be set at the start of a flow , process or another flow) and choose allow multiple values since its a collection


  4. Loop through each of the collection elements in the Opportunity contact role collection .


  5. Now create an action to send the email alert.Used the email alert created earlier.


  6. Go to page setup and create process for opportunity and set the neccessary criteria.


  7. under immediation action,you will find the collection variable from the flow .Reference it to opportunity contact roles
