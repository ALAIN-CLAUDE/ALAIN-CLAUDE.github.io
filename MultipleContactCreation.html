Use Case

Creating a component like this with all required fields can save alot of time your Sales Reps or users spend clicking to create single or individual accounts at a time

Create a new LWC

multipleContactCreation.html

Press ctr,shift,p in your VSC (windows command) and create a lwc.
Name it and Select the html file.
using lightning design system (Link : )you can select the icons you like. Now we need to do the following

  1. Render a list of elements
  2. A flexible grid system ( lightning-layout ) :
    A lightning-layout is a flexible grid system for arranging containers within a page or inside another container.
  3. lightning-record-edit-form :
    We use for:each to iterate through our list of elements and store them in item variable which we can now use to map our fields
  4. Layout Item
  5. lightning-input-field

multipleContactCreation.js

We need to define the functions for the buttons we created above

  1. Reset : handleReset().this function querys for all input fields and resets all to blank if user made a mistake and dont want to cancel each at a time.
  2. Submit : handleSubmit() this function querys for all input fields and resets all to blank if user made a mistake and dont want to cancel each at a time.
  3. addRow() uses concat to add row elements
  4. reportValidity : reportValidity() method returns true if the element's child controls satisfy their validation constraints. When false is returned, cancelable invalid events are fired for each invalid child and validation problems are reported to the user.

multipleContactCreation.js-meta.xml

Important for our component .
"apiVersion" represent the current api version we using.
set isExposed to true . if left as false you will not see the data
Now our targets are imporant since we use this to determine where we wanna see our component( AppPage,RecordPage,HomePage) . you can put all depending on your use of this component
