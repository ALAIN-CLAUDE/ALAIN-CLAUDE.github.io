Lightning Flow

A flow is the part of Lightning Flow that collects data and performs actions in your Salesforce org or an external system. Types of Lightning Flow are:

  • screen flows
  • autolaunched flows
  • Scheduled Triggered Flow
  • Record Triggered Flow
  • Platform Triggered Event flow

Project : Cpq and Invocable apex

CASE : This method can be used to perform calculations on Order object or quote based on certain fields when order is created or updated using Invocable Apex and Process to meet the requirements below..

  1. There are 11 amount fields
    • Y1, Y2, Y3, Y4 , Y5
    • Y1 Fees, Y2 Fees, Y3 Fees, Y4 Fees , Y5 Fees
    • Total Fees
  2. There are 5 boolean fields
    • Has value y1, Has value y2, Has value y3.......
  3. 2 picklist fields with value(Yes and No)
    • Multiyear-year Fee Increase
    • Multiyear-year Uniform Increase
  4. And percentage Field (Multiyear-Year fee Increase Percent)
Requirement: These checkboxes are true if the Amount fields Y1 to Y5 above have values.
  1. Write a class with an invocable method
  2. Always populate Y1 fees with value y1
  3. If Multiyear-year Fee Increase = yes and Multiyear-year Uniform Increase = yes and Multiyear-Year fee Increase Percent > 0 , check
    • if checkbox field Has value y2 is true, populate Y2 Fees field with ( percentage field * y1 + y1)
    • if checkbox Has value y3 is true, populate Y3 Fees field with ( percentage field * y2 + y2)
    • if checkbox Has value y4 is true, populate Y4 Fees field with ( percentage field * y3 + y3)
  4. populate Total Fees with sum of Y1 fees field to Y5 fees if available

  1. Create Apex Class with invocable apex method OrderUpdate. Assign @InvocableMethod to make it invocable.Create a new Order list for updating.
  2. Query for all required fields from Order whose id is in the ids list , loop over the list, calculate percent, implement field criteria with if statement and update the order passed .

  3. Go to setup and Create a new process using Process builder.this will enable passing Object Id and updating the boolean fields Has Value y1 to Has value y5.

    • Choose order as object to start the process and no criteria.For action choose Apex class and find the invocable class created. For Set Apex Variables,choose ids from the list of id of the invocable method and reference it to the order id.
    • check if Amount fields Y1 to Y2 is not null,if true update checkbox field has value Y1 to has value Y5
    • Immediate action to check Has value Y1 to Y5 to true
    • Activate
