News App Project

Using Rest Callout and Lwc to get latest news


  1. Create a Conected App

    • Go to Setup,App Manager and New connected App
    • Fill in the required fields(name,contact email). In our case Ev Branches App
    • Enable OAuth Settings
    • Provide a callback Url.Use the url of the headquarter Org as in this example
    • Select OAuth Scopes.This determines the type of access you want to give to the third party api and to this environment. Click save and continue. Within 2-10 minutes changes will take effect on server
      •
    • Our consumer key and consumer secret are automatically generated. We will use these in our Main Org(primary org)

  2. Javascript code LWC


    • Import retriveNews from your apex class
    • connectedCallBack :
    • In fetchNews , the then() method takes 2 arguments and returns a promise which are callback fxns for success or error
    • Once we receive our response(array of articles),we want to format our data(date,name and id).So we pass it as a parameter in our formatNewsData and use map fxn to get each array item and index to generate a new id for each item. Apart from id, we need the source name of each article item .Return all our properties and store in result array so we can use to render data in our html.
    • In our showModal we get each item id and store in id variable. we then check if the id is same as the id of the element selected in our result news items and if same we store in our selectedNews property display in our modal.Now with this selectedNews we can retrieve and display the news title,source name , content , author and date. put showModal property to true so each time user clicks on the button it shows and closeModal to false.
    • For get modalClass, if modal is open append slds-modal ${this.isModalOpen else "slds-fade-in-open" :""
    • get modalBackdropClass will be true only when modal is open to kmake background slightly dark

  3. HTML


    • lightning design system :
      Use grid system provides a flexible, mobile-first, device-agnostic layout system. It controls alignment, order, flow, and gutters. Keep cards in center with justify-center
    • for:each to iterate over the results and store each item in "item" variable. Fields to display item-url,date of item and title
    • Modals :
      Use Modals to display news content in a layer above the app
